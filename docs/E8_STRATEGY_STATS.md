## Strategy Components

### Entry Signals (Scoring System)
**Minimum score: 4.0** (out of 10)

Points breakdown:
- **ADX (trend strength)**: 0-2 points
  - ADX > 25: 2 points
  - ADX > 20: 1 point
- **RSI (momentum)**: 0-3 points
  - 30-40 (oversold zone): 3 points
  - 60-70 (overbought zone): 3 points
  - 40-60 (neutral): 1 point
- **MACD (trend direction)**: 0-3 points
  - Bullish crossover: 3 points
  - Bearish crossover: 3 points
  - Above signal: 1 point
- **ATR (volatility)**: 0-2 points
  - 0.5-1.5% range: 2 points
  - Outside range: 0 points

### Position Sizing Formula
```
Base units = (Account × Risk%) / Stop Distance
Adjusted units = Base units × 80% (E8 reduction)
Min size = 10,000 units (1 mini lot)
Max size = 1,000,000 units (10 standard lots)
```

### Exit Rules
- **Take profit**: +2.5% from entry
- **Stop loss**: -1.0% from entry
- **Trailing stop**: Breakeven at +1.25%, trail at +1.875%
- **Time stop**: Close if open > 7 days (trend died)
