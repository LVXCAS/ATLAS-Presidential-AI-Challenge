TROUBLESHOOTING CHECKLIST:
===========================

IF SCANNER NOT RUNNING:
□ Run CHECK_SCANNER_STATUS.bat
□ Check Task Scheduler - tasks enabled?
□ Was PC on at 6:30 AM PT?
□ Run START_TRADING.bat manually to test
□ Check VIEW_LATEST_LOG.bat for errors

IF NO TRADES EXECUTED:
□ Check auto_scanner_status.json - trades_today count
□ Review logs - were opportunities found?
□ Verify score threshold (must be >= 8.0)
□ Check daily limit (max 4 trades/day)
□ Confirm market hours (6:30 AM - 1:00 PM PT, M-F)

IF PYTHON ERRORS:
□ Run TEST_SETUP.bat
□ Check Python version: python --version
□ Reinstall packages: pip install -r requirements.txt
□ Verify .env file has API keys

IF TASK SCHEDULER ERRORS:
□ Re-run SETUP_AUTOMATED_STARTUP.bat as administrator
□ Check Windows Event Viewer for task errors
□ Verify user account has necessary permissions


EMERGENCY PROCEDURES:
=====================

TO STOP ALL TRADING IMMEDIATELY:
□ Double-click EMERGENCY_STOP.bat
□ Verify all processes killed
□ Check scheduled tasks disabled
□ Review emergency_stop.flag created

TO RESUME AFTER EMERGENCY STOP:
□ Delete emergency_stop.flag file
□ Run SETUP_AUTOMATED_STARTUP.bat as administrator
□ Run CHECK_SCANNER_STATUS.bat to verify
□ Optionally run START_TRADING.bat to test
