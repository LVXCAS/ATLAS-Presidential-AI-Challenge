## RISK MANAGEMENT (THE $600 LESSON APPLIED)

### 1. Position Sizing Rules

```python
# Never risk more than 4% on a single trade
max_risk_per_trade = account_balance * 0.04

# For options, this means:
If buying $5 contracts:
  max_contracts = (max_risk_per_trade) / ($5 × 100)

Example with $4k account:
  max_risk = $4k × 0.04 = $160
  If option costs $5/contract: max 3 contracts ($160 / $50 = 3.2)
```

### 2. Daily Loss Limit (Learned from Forex)

```python
# Stop trading if daily loss exceeds 5% of account
daily_loss_limit = account_balance * 0.05

# With $4k account: $200 daily loss limit
# Hit this → stop for the day, reassess tomorrow
```

### 3. Monthly Drawdown Limit

```python
# If month-to-date loss exceeds 15%, reduce position sizing by 50%

```

```python

```

```

Entry: Last 30 min of trading day
Exit: Next day bounce (usually 1-2%)
Win rate: 60%
Example: SPY drops $480 → $470, buy $475 calls, exit at $475-480
```

**3. Trend Continuation Pullbacks**
```
Setup: Strong uptrend, pullback to 20 EMA
Strategy: Buy 2-4 week calls at support
Entry: RSI <50, bouncing off 20 EMA
Exit: Target resistance or 2:1 R:R
Win rate: 55%
Example: AAPL uptrend, pulls back $175 → $172 (20 EMA), buy $175 calls
```

```
Win rate: 52%
```

**5. Breakout Confirmation**

```
Win rate: 48% but 3:1 R:R makes it profitable
```

## MULTI-AGENT SYSTEM INTEGRATION

### Your 10 Agents and Their Roles:

**Market Analysis Expert:**
- Scans 20 symbols for technical setups
- Identifies: Breakouts, pullbacks, reversals, earnings momentum
- Outputs: Setup quality score (1-10)

**Risk Assessment Expert:**
- Calculates position size based on account balance
- Monitors daily P/L, checks against limits
- Outputs: Max contracts allowed, risk/reward ratio

**Options Strategy Expert:**
- Selects optimal strategy (long call, spread, iron condor, etc.)
- Based on: IV percentile, time to expiration, conviction level
- Outputs: Strategy type, strike prices, expiration

**Sentiment Analysis Expert:**
- Analyzes news, social media (Twitter, Reddit WSB)
- Detects: Bullish/bearish sentiment shifts
- Outputs: Sentiment score (-10 to +10)

**Trade Execution Expert:**
- Places orders via Alpaca API
- Optimizes: Entry timing, limit vs market orders
- Monitors: Fill quality, slippage

**Performance Tracking Expert:**
- Logs all trades (entry, exit, P/L, win/loss)
- Calculates: Win rate, ROI, Sharpe ratio, max drawdown
- Outputs: Daily/weekly/monthly reports

**Portfolio Optimization Expert:**
- Ensures diversification (not all positions same direction)
- Manages: Position correlation, sector exposure
- Outputs: Portfolio health score

**Macro Economic Expert:**
- Tracks: Fed policy, economic data (CPI, jobs, GDP)
- Adjusts strategy based on macro regime
- Outputs: Market regime (bullish, bearish, neutral, volatile)

**High Frequency Expert:**
- Monitors: Intraday momentum, unusual volume, order flow
- Detects: Short-term (15-60 min) opportunities
- Outputs: Scalping opportunities (optional, can disable)

**Crypto Market Expert:**
- (Can be disabled for options-only deployment)
- OR adapted to track crypto correlation with tech stocks
