SETUP (ONE TIME - 2 MINUTES):
------------------------------

1. Right-click "SETUP_AUTOMATED_STARTUP.bat"
   Select "Run as administrator"
   Wait for "SETUP COMPLETE" message

2. Double-click "CHECK_SCANNER_STATUS.bat"
   Verify scheduled task is created

3. DONE! Scanner will now run automatically every day at 6:30 AM PT


DAILY USE:
----------

Morning:
  - Scanner starts automatically at 6:30 AM PT
  - Scans market for opportunities
  - Executes trades if score >= 8.0
  - You don't have to do anything!

Optional Check:
  - Run "CHECK_SCANNER_STATUS.bat" to verify it's running
  - Run "VIEW_LATEST_LOG.bat" to see what happened


MANUAL CONTROL:
---------------

Start Now:       Double-click "START_TRADING.bat"
Stop Scanner:    Double-click "STOP_SCANNER.bat"
Check Status:    Double-click "CHECK_SCANNER_STATUS.bat"
View Logs:       Double-click "VIEW_LATEST_LOG.bat"
Emergency Stop:  Double-click "EMERGENCY_STOP.bat"


WHAT IT DOES:
-------------

✓ Runs every day at 6:30 AM PT (market open)
✓ Scans S&P 500 for options opportunities
✓ Auto-executes high-scoring trades (score >= 8.0)
✓ Max 4 trades per day (safety limit)
✓ Only trades during market hours (M-F, 6:30 AM - 1:00 PM PT)
✓ Logs everything to logs/ directory
✓ Survives system reboots
✓ Handles errors gracefully

TROUBLESHOOTING:
----------------

Scanner not running?
  → Run CHECK_SCANNER_STATUS.bat
  → Run SETUP_AUTOMATED_STARTUP.bat again
  → Check VIEW_LATEST_LOG.bat for errors

No trades executed?
  → Normal if no good opportunities (score < 8.0)
  → Check daily limit (max 4 trades/day)
  → Verify market hours (6:30 AM - 1:00 PM PT, M-F)

Need help?
  → Read AUTOMATED_STARTUP_README.md
  → Check logs/ directory
  → Verify Python and dependencies installed
